VIRTUAL ENVIRONMENT DETAILS
The folder todo is the virtual environment variable used to run the application

TO RUN THE APP
1. EXTRACT THE ZIP FILE INTO DESKTOP
2. OPEN COMMAND PROMPT
3. RUN THE COMMAND cd Desktop
5. RUN THE COMMAND cd djangotodoapp
6. RUN THE COMMAND pip install virtualenv TO INSTALL A VIRTUAL ENVIRONMENT
7. RUN THE COMMAND virtualenv todoenv TO CREATE THE VIRTUAL ENVIRONMENT
6. RUN THE COMMAND todoenv\scripts\activate TO ACTIVATE VIRTUAL ENVIRONMENT
7. RUN THE COMMAND python manage.py runserver TO RUN THE WEB SERVER
8. GO TO ANY BROWSER AND PASTE THE LINK DISPLAYED IN THE TERMINAL
